# PremiumWeb — Эксперт по автоматизации

Премиальный сайт для эксперта по автоматизации с темной/светлой темой, модальными окнами и адаптивным дизайном.

## 🚀 Демо
- **Vercel:** https://site6.vercel.app
- **GitHub Pages:** https://b8k13mus.github.io/site6/

## 🛠 Технологии
- **HTML5** — семантическая разметка
- **CSS3** — кастомные свойства, градиенты, glass-morphism
- **Vanilla JavaScript** — без фреймворков, быстрый и легкий
- **Vercel** — хостинг с глобальной CDN и автоматическими деплоями
